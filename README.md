# 🌐 Student Registration & Login System

This project is a simple *Student Registration and Login System* built using HTML, CSS, Bootstrap, and PHP.  
It allows users to create an account and log in securely.

🔗 Live Demo: https://piyush0kumar.github.io/project2/

---

## 📌 Features

- 📝 User Registration Form  
- 🔐 Login Authentication  
- 📱 Responsive Design (Mobile + Desktop)  
- 🎨 Clean UI using Bootstrap  
- ⚡ Fast and lightweight  

---

## 🛠️ Technologies Used

- *Frontend:*
  - HTML5
  - CSS3
  - Bootstrap
  - Font Awesome

- *Backend:*
  - PHP

- *Database:*
  - MySQL (via phpMyAdmin)
